Простой пример построения дерева решений
==========

Строим дерево фигур на основе эталонного набора. Входные данные это набор любых
признаков и класса к которому они принадлежат. На выходе готовое дерево
выводится в консоль

## Как работать с этим:

Установить lua 5.1 

Ввести в main.lua переменную etalon следующего вида:
```
{ class = нужный класс, признак_1 = 0, признак_2 = 0, признак_3 = 0, признак_4 = 0 },

```

Ввести в командной строке:
```
lua main.lua 
```

Результат:
```
Ваше дерево слева-направо
```
## Алгоритм работы:

0. Берем все множество признаков и находим признак с наибольшим количеством
   информативности.
1. Разбиваем все на подмножества результатов с этим признаком.
2. Проходимся рекурсивно по подмножествам, убрав из них признак из 0 шага.
3. Выводим результат.

## Проблемы которые не решены:

0. Вывод результатов так себе
1. Из детей дерева не убираются классы которые классифицируются где-то выше.
   Тогда получилось бы меньшее количество спусков
